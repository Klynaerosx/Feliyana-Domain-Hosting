What's the definition of Domain?

A domain name is a human-friendly website address on the Internet, like google.com or wikipedia.org. It replaces complex numerical IP addresses. By typing words instead of numbers into a browser, people can quickly reach websites and online services. (Hostinger.com)

Domain Extensions

Domain Extensions and Its Types?

Internet Assigned Authority (IANA) classifies three types of domains. First, Country code top level domain (ccTLD), used for country code, consists of 2 letters. For example .jp for Japan, and .id for Indonesia. Second, Generic Top Level Domain, used for various organisations, consisting of 3 letters or more, for example .com for commercial organisations. Third, Infrastructure top level domain, for example .arpa. Other examples:

.Gov

Used for government

.Edu

Used for educational purpose

.Org

Used for non-profit organisations/activivities

.Mill

Used for military

.Com

Used for profit/commercial organisation

.Net

Used for network organisation

.Name

Used for personal/fammily

.Tv

Used for television

.Info

Used for information purposes

.Biz

Used for business purpose

.Travel

Used for tourism

Definition Of Hosting

Definition of Hosting & Its Function?

PHosting is a place used as a media to store all of the data related to a website to be accessed and managed by the internet. The data or website file that is stored are photo, video, script, application, plugin, and so on. It is based on the website content.-The function of hosting is to manage and run the server, implement security measures to protect the website, and ensure that data such as text, photos, and other website files can be served in the visitor's web browser.
